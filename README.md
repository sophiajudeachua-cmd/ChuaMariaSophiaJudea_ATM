# ChuaMariaSophiaJudea_ATM
A console-based ATM simulation featuring balance checks, deposits, withdrawals, and a persistent session loop.

# Description
This project is a simulated Automated Teller Machine (ATM) application. It allows users to manage a virtual balance through a persistent menu loop, handling deposits and withdrawals with built-in logic to prevent overdrafts.

# File Structure
app.py – Main banking application script.

README.md – Project documentation.

# How to Run the Application Locally
1. Make sure Python is installed.

2. Open a terminal in the project folder.

3. Run the application: python app.py

4. Use the numeric menu (1-4) to navigate through your banking options.
